# Exo

### 1. Création des entités

Créez 2 entités :

**Produit** : id, nom, prix

**Categorie** : id, nom

### 2. Lien entre les entités

Un **Produit** peut avoir une **Catégorie**, une **Catégorie** peut être liée à plusieurs **Produit**

### 3. Ajout, visualisation, modification de Categorie

Ecrivez l'ajout, visualisation, modification de l'entité Catégorie

### 4. Ajout, visualisation, modification de Produit

Ecrivez l'ajout, visualisation, modification de l'entité Produit, on doit pouvoir associer une catégorie lors de la création / modification
